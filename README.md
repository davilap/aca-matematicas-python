# ACA Matemáticas – Ingeniería de Sistemas (Python)

Este repositorio contiene el desarrollo del ejercicio básico de la Actividad de Construcción Aplicada (ACA) de Matemáticas.
Se implementó un mini buscador de registros en Python con datos sintéticos, integrando:
- Álgebra booleana (filtros con AND / OR / NOT y validación de De Morgan)
- Algoritmos (búsqueda lineal y binaria, ordenamiento con `sorted()`, algoritmo de Euclides)
- Estructuras de datos (Árbol Binario de Búsqueda - BST para consultas por `id`)

## Contenido del repositorio
- `aca.py`: script principal con generación de datos, filtros, algoritmos, BST y experimento.

- Windows / macOS / Linux
- Python 3.8+ (recomendado)
